# SabbFlow Bot

Telegram bot for SabbFlow heating calculator.

## Environment Variables
- `BOT_TOKEN` — Telegram bot token from @BotFather
- `WEBAPP_URL` — Mini App URL (default: https://saidislom6134-cmyk.github.io/SabbFlow-/)

## Deploy on Railway
1. Connect this repo to Railway
2. Add environment variables
3. Deploy!
